Fractus Client

This is what you need to connect to the Fractus network. If you don't know what you're trying to download, this is probably what you want. Download this from Github at:
http://www.github.com/bowenl2/Fractus

Fractus Server

The Server will allow clients to authenticate one another and establish each others' locations. It's probably better that you use the main one at fract.us but if you have a compelling reason to host your own, the source code is provided below. There are no installation instructions and several dependencies such as MySQL. The database schema (but obviously not the data) is provided.
http://www.github.com/bowenl2/FractusServer
